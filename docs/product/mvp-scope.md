# MVP Scope

## In Scope

- Telegram bot intake
- requirement capture, plan drafting, and explicit approval flow
- quick acknowledgement with plan details and rough estimate
- background job lifecycle model with proactive progress updates
- code generation, debugging, and explanation workflows
- YAML config with local default and explicit override path
- role-based provider selection with provider URLs
- optional parallel fan-out across multiple configured providers for one agent
- installable Python package layout
- local SQLite persistence for runs, chat history, and learning excerpts

## Out Of Scope

- arbitrary code execution sandbox
- advanced multi-user admin workflows
- distributed queue infrastructure
